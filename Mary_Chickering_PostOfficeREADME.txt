mary_chickering_postoffice

The user enters their package dimensions and zip code then the code produces the total cost of mail. The program finds the zones of each zip based on certain parameters and then subtracts the two zones to find the total zone. Then calculates the cost associated with the total zone and the package type.

Input: Input package height, width, thickness, zip code 1 and zip code 2
Output: Total cost to mail

Sample input: 4, 4, .009, 02893, 08516
Sample output: .23
